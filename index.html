Projects

LeafSense

LeafSense is an intelligent plant disease detection system designed for crops like strawberry, coffee, and tea. It uses computer vision to identify plant diseases from images in real time, combined with an educational platform offering CRUD access to plant care articles. The system is powered by Firebase and deployed on Google Cloud Run, with integrations to Cloud SQL and Cloud Storage for efficient backend operations.

LuniXhar

LunixHAR is a real-time Human Action Recognition (HAR) system that combines YOLOv8 for person detection, MobileNetV2 for feature extraction, and a Transformer-based model for action classification. It supports multi-person tracking with unique IDs using ByteTrack, and processes webcam streams via a local API. Designed for deployment on Google Cloud Run, LunixHAR enables robust, real-time activity monitoring from multiple camera inputs.
